Product Class Project
====================

Description:
------------
This project demonstrates a simple C++ class called 'Product' with:
- Private attributes: id, name, price
- Constructor to initialize these attributes
- Destructor that prints a message when the object is destroyed
- Method 'printDetails()' to display product information


Compilation Instructions:
-------------------------
Using g++ in terminal or command prompt:

Linux / Mac:
g++ main.cpp  -o program
./program

Windows (MinGW):
g++ main.cpp  -o program.exe
program.exe

Expected Output:
----------------
Product ID: 101
Product Name: Laptop
Product Price: $999.99
Destructor called for product: Laptop
